# Etch-A-Sketch

A digital replica of the classic Etch-A-Sketch toy, built as part of [The Odin Project](https://www.theodinproject.com) curriculum. 

## Features
- **Dynamic Grid Size**: Adjust the resolution of the drawing board using the slider.
- **Interactive Knobs**: The physical knobs rotate as you draw, mimicking the mechanical feel of the original toy.
- **Shake to Clear**: Click the "Shake to Clear" button to reset your drawing with a custom animation.
- **Responsive Design**: Styled to look like it's sitting on a wooden desk, with scaling for mobile devices.

## Technologies Used
- HTML5
- CSS3 
- JavaScript
